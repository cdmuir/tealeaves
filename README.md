## Get leafevolver
From CRAN (not on CRAN yet)

```r
install.packages("leafevolver")
```

or from rOpenSci

```r
install.packages("devtools")
devtools::install_github("cdmuir/leafevolver")
```

And load leafevolver

```r
library("leafevolver")
```
