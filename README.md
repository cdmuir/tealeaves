## Get tealeaves
From CRAN (not on CRAN yet)

```r
install.packages("tealeaves")
```

or from GitHub

```r
install.packages("devtools")
devtools::install_github("cdmuir/tealeaves")
```

And load tealeaves

```r
library("tealeaves")
```
